## Статус ввода

Чтобы указать, что пользователь указал верные или, наоборот, неверные данные
необходимо передать в компонент атрибуты **valid** или **invalid** соответственно.

<Playground>
  <div className="margins">
    <Select valid>
      <option>🍏Apple</option>
      <option>🍊Orange</option>
      <option>🍉Melon</option>
    </Select>
    <Select invalid>
      <option>🍏Apple</option>
      <option>🍊Orange</option>
      <option>🍉Melon</option>
    </Select>
  </div>
</Playground>

## Размеры

В свойстве компонента **size** можно указать один из трех размеров: _small_, _normal_ и _big_.

По умолчанию кнопка принимает нормальный размер.

<Playground>
  <div className="margins">
    <Select theme="primary" size="small">
      <option>🍏Apple</option>
      <option>🍊Orange</option>
      <option>🍉Melon</option>
    </Select>
    <Select theme="primary" size="normal">
      <option>🍏Apple</option>
      <option>🍊Orange</option>
      <option>🍉Melon</option>
    </Select>
    <Select theme="primary" size="big">
      <option>🍏Apple</option>
      <option>🍊Orange</option>
      <option>🍉Melon</option>
    </Select>
  </div>
</Playground>
